# ТЗ MVP: веб-апп для учета лотов и расчета пайда

## Цель
Сделать простую PWA веб-аппку для одного пользователя: вести заявки по лотам, отдельно вести расчеты, видеть краткий dashboard и хранить данные локально в браузере.

## Разделы

### 1. Dashboard
- Количество лотов: все заявки.
- Срок истекает: 3 ближайших активных заявки, показывать только номер лота и срок.
- Выручка за месяц: сумма `ЦП` по заявкам со статусом `Договор` или `Закрыто`, где дата закрытия попадает в текущий месяц.

### 2. Заявки
Поля:
- Номер лота
- Статус
- Наименование
- Категория
- Срок с временем
- Адрес
- Бюджет
- Пайда %
- Комментарий
- Ссылка
- Файлы/документы
- Дата закрытия

Статусы:
- Проект
- Анализ
- Подали
- Выиграли
- Проиграли
- Договор
- Отказ
- Закрыто

### 3. Есеп
Поля:
- Номер лота
- Закуп
- Налог
- Доставка
- Работа
- Доп.расход
- ЦП
- Пайда
- Пайда %

Формулы:
- `Налог = ЦП * 10%`
- `Пайда = ЦП - Закуп - Налог - Доставка - Работа - Доп.расход`
- `Пайда % = Пайда / ЦП * 100`

## Первая версия
- Работает без сервера.
- Данные сохраняются в браузере.
- Можно добавить, изменить и удалить заявку.
- Можно добавить, изменить и удалить расчет.
- В `Есеп` номер лота выбирается из заявок.
- Есть экспорт и импорт резервной копии в JSON.
- Есть PWA manifest и service worker.
